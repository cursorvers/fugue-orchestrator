## 7. Adapter Contract

Adapter files (`CLAUDE.md`, `CODEX.md`) must contain only:
- Role-specific deltas that cannot live in SSOT.
- Pointers to this file and a minimal command reference.
- No duplicated long policy text from SSOT.
